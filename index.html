One request, wrapped for delivery

      
Follow a single request as it travels left → right. Each stop wraps it in one more header, so the data grows a new coloured block.

      
The bright core is the real data the app cares about. Watch it shrink to a sliver as delivery metadata piles on — that ratio is the whole idea.

      
Scroll or swipe to move the data. Each node's colour is the header it adds. Tap the data to see its real headers, as a tool like tcpdump would show them.
